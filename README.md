A collection of my [POV-Ray](https://www.povray.org/) ray-tracing scenes, created in the early 2000s. Some are entries to the [Internet RayTracing Contest (IRTC)](http://www.irtc.org/); others are personal experiments.

Each folder contains the `.pov` source file(s) and a `run.sh` script to render the scene.
